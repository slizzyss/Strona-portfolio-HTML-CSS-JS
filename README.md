# Strona Portfolio – HTML/CSS/JS

Nowoczesna strona portfolio zakodowana od zera w czystym HTML, CSS i JavaScript.

## Co zawiera:
- Dark theme z animacjami CSS
- Sekcja hero z efektami blob i noise
- Karty usług i projektów
- Sekcja kontaktowa
- W pełni responsywna (mobile/desktop)

## Technologie:
- HTML5
- CSS3 (animacje, zmienne, grid, flexbox)
- Google Fonts (Syne + DM Mono)

## Podgląd:
Otwórz plik `portfolio.html` w przeglądarce.
